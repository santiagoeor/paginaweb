Api Rest Laravel De Pendientes

Durante este proyecto, exploré a fondo la creación de una API en Laravel, abordando aspectos clave como migraciones para definir esquemas de bases de datos y establecer relaciones entre tablas. Utilicé MySQL como base de datos y aprendí a implementar un sistema de autenticación basado en JWT para la seguridad de inicio de sesión.

Creé modelos y controladores que permiten realizar operaciones CRUD (Crear, Leer, Actualizar, Eliminar) para usuarios, categorías y pendientes. Además, diseñé endpoints con métodos HTTP (GET, POST, PUT, DELETE) para gestionar eficientemente la información a través de la API.

Este proyecto destacó la aplicación práctica de conceptos avanzados en Laravel y la construcción de una API robusta y funcional.

Cliente De Pendientes En Angular

En este proyecto, construí el frontend para interactuar con mi API de pendientes en Laravel mediante Angular. Aplicando los conocimientos adquiridos en un curso de Angular, implementé un interceptor para enviar el token obtenido durante el inicio de sesión al backend. Además, establecí un guardian para proteger las rutas y, por defecto, convertí la aplicación en una PWA (Progressive Web App).

Optimicé el rendimiento utilizando Lazy-loading para cargar módulos de usuarios, categorías y pendientes de manera eficiente. Implementé indicadores de carga (loading) para mejorar la experiencia del usuario al cargar, crear, editar y eliminar usuarios, categorías y pendientes.

Este proyecto demostró mi capacidad para aplicar prácticas avanzadas en Angular, asegurando un consumo eficiente de la API desarrollada en Laravel y proporcionando una experiencia de usuario fluida y moderna.
