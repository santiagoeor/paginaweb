Biografia personal

¡Hola, soy Santiago! Soy un entusiasta programador web con experiencia en proyectos que abarcan diversas tecnologías, como HTML, PHP, CSS, JavaScript, Bootstrap, MySQL, Go, Python (Django y Flask), y Laravel. En el 2023, desarrollé una aplicación PWA utilizando Angular y Laravel, donde implementé JSON Web Tokens para el inicio de sesión y la seguridad de las rutas en el backend. Mi constante búsqueda de desafíos y mi compromiso con el aprendizaje continuo me impulsan a crecer como profesional en el emocionante campo del desarrollo web.
